Treasury Management

v0.9 Beta

Real-time treasury intelligence dashboard for corporate funding desks. Monitors short-term yields, overnight rates, FX cross-rates, and macro catalysts in a single institutional-grade view.

Key Features
  • Grouped bar chart: 1M/3M/6M yields vs T-7 and T-14
  • Live funding rates: SOFR, EFFR, T-Bill benchmarks
  • FX converter with cross-rates, pips, and quick treasury pairs
  • TradingView economic calendar (US, high importance)
  • Risk pills: VIX, DXY, IG OAS, 10Y yield at a glance
  • Bloomberg TV live stream + WSJ news feed
  • Rate movement alerts when yields shift >5 bps vs T-7
Technology

Vanilla JS · Chart.js · Cloudflare Workers · FRED API · NY Fed API · Yahoo Finance · GitHub Pages
